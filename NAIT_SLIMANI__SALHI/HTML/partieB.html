ViaMin

Dans la deuxieme partie du projet on veut implementer un algrithme efficace pour minimiser le nimbre de via dans le cercuit pour cela on a decidé de presenter des relation entre les extrimité des segments (points ) et entre les segments interectant et un graphe non orienté c'set la meilleure façon de representer ces relations.
Pour minimiser les via on va detecter les cycles impairs dans ce graphe et et on ajoute un via pour chaque cycle impaire detecté.
